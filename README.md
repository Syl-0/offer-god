# Offer 来了 — BOSS 直聘求职助手

一款基于 AI 和传统命理学的Chrome浏览器扩展，帮助求职者提供针对性的求职建议以及岗位匹配度评估。

## ✨ 功能特性

### 👤 求职者画像生成
- **智能简历解析**：上传 PDF 简历，自动提取核心技能、学历背景、工作年限等关键信息
- **硬性特质识别**：智能识别 985/211 院校、学历层次、英语水平、毕业年份等硬性条件
- **软性特质分析**：分析 AI/大模型经验、产品能力、项目管理、团队管理、大厂背景等软实力
- **八字排盘**：根据出生时间自动计算四柱八字、五行力量、大运流年

### 📊 匹配度分析
- **科学维度评分**：基于简历关键词与 JD 的语义匹配，精准评估岗位科学匹配度
- **岗位差距分析**：清晰展示您的优势与不足，对比岗位要求
- **五行职业匹配**：分析日主五行与各行业岗位的生克关系，给出玄学维度匹配度

### 💡 深度求职建议
- **综合分析**：结合岗位要求、公司特点、匹配分数，给出整体求职策略
- **简历优化**：针对具体岗位的简历调整建议，突出相关经验
- **面试准备**：给出针对性、可执行的面试准备策略
- **注意事项**：结合公司规模背景，给出需要特别留意的问题和风险提示

## 📦 安装

### 方式一：直接安装（推荐）

1. 前往 [Releases](https://github.com/Syl-0/offer-god/releases) 页面
2. 下载最新版本的 `offer-god.zip`
3. 解压到任意目录
4. 在 Chrome 中加载：
   - 打开 `chrome://extensions/`
   - 开启右上角「开发者模式」
   - 点击「加载已解压的扩展程序」
   - 选择解压后的目录

### 方式二：从源码构建

适合想自定义修改的开发者：

```bash
git clone https://github.com/Syl-0/offer-god.git
cd offer-god
npm install
npm run build
```

然后在 Chrome 中加载 `dist` 目录。

## 🚀 使用方法

### 1. 配置大模型 API

首次使用需要配置 LLM API：

1. 点击扩展图标，进入设置页面
2. 勾选「我已阅读并理解上述说明」
3. 在「大模型配置」区域选择模型服务商：
   - OpenAI（推荐 GPT-4o-mini）
   - Claude (Anthropic)
   - 智谱 GLM
   - DeepSeek
   - 阿里千问
   - 百度文心
   - Moonshot
   - 或自定义 API
4. 输入 API Key
5. 点击「测试连接」验证配置
6. 点击「保存」

### 2. 上传简历

1. 在设置页面上传 PDF 格式简历
2. 系统会自动解析并提取关键信息

### 3. 填写出生信息（可选）

如需玄学维度分析：
1. 填写出生年月日时
2. 选择性别（用于排大运方向）
3. 填写出生地（用于真太阳时校正）

### 4. 开始使用

在 BOSS 直聘网站上浏览职位时：
- 列表页：每个职位卡片会显示匹配度分数
- 详情页：可查看详细的匹配分析和求职建议


## ⚙️ 配置说明

### 权重调整

可在设置页面调整科学与玄学维度的权重比例：
- 科学权重：控制简历匹配度的占比
- 玄学权重：控制八字匹配度的占比

### 暂停功能

如需临时关闭扩展功能，可勾选「暂停在 zhipin.com 显示标签」。
